# 🚀 Smart Notes API

A secure and intelligent Notes Management REST API built using **Spring Boot**, **Spring Security**, **JWT Authentication**, **PostgreSQL**, and **Spring AI**.

## ✨ Features

### 🔐 Authentication & Security

* JWT Authentication
* Refresh Token Support
* Secure User Registration & Login
* Spring Security Integration
* Role-Based Security Ready

### 📝 Notes Management

* Create Notes
* Update Notes
* Delete Notes
* View Personal Notes
* User-specific Note Access

### 🤖 AI Features

* AI-Powered Note Summarization
* Smart Content Processing
* Gemini/OpenAI Compatible Integration via Spring AI

### 🗄️ Database

* PostgreSQL Integration
* JPA/Hibernate ORM
* Automatic Table Management

## 🔑 Environment Variables

Create a `.env` file and configure:

DB_URL=your_database_url

DB_USERNAME=your_database_username

DB_PASSWORD=your_database_password

OPENAI_API_KEY=your_api_key

---

## ▶️ Run Project

1. Clone Repository

git clone https://github.com/your-username/smart-notes-api.git

2. Configure Environment Variables

3. Run Application

mvn spring-boot:run
